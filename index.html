Connect Google Sheets

Your data will sync across all your devices via a private Google Sheet. Follow these steps once:

1. Go to sheets.google.com and create a new blank spreadsheet
2. Click Extensions → Apps Script. Delete any existing code and paste in the contents of Code.gs
3. Click Deploy → New deployment. Choose type Web app. Set Who has access to Anyone. Click Deploy and authorise
4. Copy the Web app URL and paste it below
